<h1 style="text-align:center;">Cybersecurity & Networking Tools</h1>

<div class="tools-grid">

<div class="tool-box">
<h3>IPInfo</h3>
<p>Lookup IP address information such as ISP, ASN, geolocation, and network ownership.</p>
<a href="https://ipinfo.io" target="_blank">Open Tool</a>
</div>

<div class="tool-box">
<h3>AbuseIPDB</h3>
<p>Check whether an IP address has been reported for malicious activity.</p>
<a href="https://abuseipdb.com" target="_blank">Open Tool</a>
</div>

<div class="tool-box">
<h3>VirusTotal</h3>
<p>Analyze suspicious files, URLs, domains, and IP addresses using multiple engines.</p>
<a href="https://virustotal.com" target="_blank">Open Tool</a>
</div>

<div class="tool-box">
<h3>Shodan</h3>
<p>Search for internet connected devices and open services across the internet.</p>
<a href="https://shodan.io" target="_blank">Open Tool</a>
</div>

<div class="tool-box">
<h3>DNS Checker</h3>
<p>Verify DNS records and propagation across global DNS servers.</p>
<a href="https://dnschecker.org" target="_blank">Open Tool</a>
</div>

<div class="tool-box">
<h3>crt.sh</h3>
<p>Search SSL/TLS certificates issued for domains using certificate transparency logs.</p>
<a href="https://crt.sh" target="_blank">Open Tool</a>
</div>

</div>
